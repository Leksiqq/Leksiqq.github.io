Rubik2Console

О программе

Приложение позволяет расчитывать состояния кубика Рубика 2×2×2 при различных поворотах, вычислять повороты, необходимые для перевода из некоторого начального в некоторое конечное состояние, а также для сборки. Подробнее...

Установить

net8.0-windows (x64) Rubik2Console
net6.0-windows (x64) Rubik2Console
